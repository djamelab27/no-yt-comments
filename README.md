# NoYTComments

Simple Chrome extension to hide YouTube comments.

## Why

Sometimes I open YouTube for one video… and end up reading comments I didn’t even want to see.

This just removes them.

## What it does

- Toggle ON / OFF
- Keeps your choice even after you close the browser
- Works on all YouTube pages

## How to use

1. Install the extension (Chrome Web Store link soon)
2. Turn it ON
3. Reload YouTube

## Privacy

No data collected.

The extension only saves your ON/OFF preference locally.
